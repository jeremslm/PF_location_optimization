# PF Coil Locations Optimization

This repository contains scripts and notebooks for PF (Poloidal Field) coil locations optimization in both fixed and free boundary scenarios, with applications to DIII-D equilibria.

## Contents
- **`PF_coil_optimization.ipynb`** – Implements the optimization scheme for PF coils in a **fixed-boundary** configuration.  
- **`DIIID_free_boundary.ipynb`** – Handles the **free-boundary** case of PF coil optimization for DIII-D.  
- **`g192185.02440`, `g174864.02500`, `g173630.03000`** – DIII-D equilibria data.  
- **`helper_functions_angle.py`** - Side functions.
- **`DIIID_geom.json, DIIID_mesh_scantest_withSolenoid.h5`, DIIID_mesh.h5** - Geometric and meshing DIII-D data.
